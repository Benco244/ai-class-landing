### Option 2: Deploy to Web Hosting

#### Using Netlify (FREE & EASIEST)
1. Go to https://www.netlify.com/
2. Sign up for free account
3. Drag and drop the entire `ai-class-landing` folder
4. Get your live URL instantly!
5. You can connect a custom domain later

#### Using Vercel (FREE)
1. Go to https://vercel.com/
2. Sign up for free
3. Import the folder or connect via GitHub
4. Deploy in seconds

#### Using GitHub Pages (FREE)
1. Create a GitHub account if you don't have one
2. Create a new repository called `ai-class-landing`
3. Upload all files
4. Go to Settings > Pages
5. Select main branch and save
6. Your site will be live at `yourusername.github.io/ai-class-landing`

#### Using Traditional Web Hosting (cPanel/FTP)
1. Log into your hosting control panel
2. Go to File Manager or use FTP client (FileZilla)
3. Upload all files to `public_html` or `www` folder
4. Access via your domain
